## 📋 Описание

Majestic Sorter — программа для автоматической сортировки скриншотов медицинской деятельности в GTA V RP на сервере Majestic RP.

Программа распознаёт:
- 💊 **Таблетки** — выдача таблеток пациентам
- 💉 **Вакцины** — вакцинация пациентов  
- 🚑 **ПМП** — первая медицинская помощь (реанимация)

И автоматически определяет больницу:
- 🏥 ELSH (Город)
- 🏜 Sandy Shores (Пригород)
- 🌊 Paleto Bay (Пригород)

## 🚀 Установка

### Способ 1: Готовый .exe (рекомендуется)
1. Скачайте последнюю версию с [GitHub Releases](https://github.com/Orange2-invalide/MadjesticRP_Sorter/releases)
2. Запустите `MajesticSorter.exe`
3. Готово!

### Способ 2: Из исходников
```bash
# Клонируйте репозиторий
git clone https://github.com/Orange2-invalide/MadjesticRP_Sorter.git
cd MadjesticRP_Sorter

# Установите зависимости
pip install -r requirements.txt

# Запустите
python main.py
```

## 📖 Как использовать

### Быстрый старт
1. **Укажите входную папку** — где лежат ваши скриншоты
2. **Укажите выходную папку** — куда складывать отсортированные
3. **Нажмите "СОРТИРОВАТЬ"**
4. Готово! Скриншоты разложены по папкам

### Структура выходных папок
```
📁 Sorted/
├── 📁 Таблетки - ELSH/
